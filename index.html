¿QUÉ ES HTML?

Es un lenguaje de marcado ya que se codifica un documento mediante etiquetas.
Se marca con hiperlinks, contiene enlaces y etiquetas, por lo que no es un lenguaje de programación.

ETIQUETAS CONTENIDAS EN HTML.

h1 a h6: Títulos indexos en la red
p: Párrafos
img: Imágenes
a: Hiperlinks

FORMATO DE ETIQUETAS

Strong (etiqueta)
Class (atributo)
Keyword (valor)
Contenido
/Strong (etiqueta de cierre)

CATEGORÍAS DE ETIQUETAS.

Textuales, agrupación, multimedia, tablas, formularios, interactivos, semánticos y scripting.

ESTRUCTURA DE HTML.

Consideremos que todas las etiquetas van escritas entre corchetes mayor y menor que.

!Doctype html (utilizamos HTML5)

html (apertura del elemento raíz)
head (cabecera)
title (título de la página)
/head (cierre del elemento raíz)

body (apertura del cuerpo del documento)
h1 - h6 (encabezados, proporcionan jerarquía de la información presente en el documento)
p (utilizado para almacenar información en párrafos)
/body (cierre del cuerpo)
/html (cierre de la página)

CSS

Es un lenguaje que le da diseño a nuestra página, fuentes, colores , márgenes, espaciado, fondos, etc

JAVASCRIPT

Se encarga de darle uso a los hipervínculos, el autorrellenado y efectos visuales en el navegador.

FRONT-END

Parte de la web visible para el usuario.

Conformado por:

HTML: Estructura y esqueleto.

CSS: Los colores, estilo y vibra.

JavaScript Interactividad y movimiento.

Algunos ejemplos de estos son:
Efectos de movimiento y brillo en los botones de la página web, formularios con autorrellenado, entre otras animaciones.

BACK-END

Se refiere a la parte lógica de nuestras páginas web, no visible para el usuario.
Este mismo sirve para:


* Guardar información en base de datos.
* Validar usuarios y envíar correos.

LENGUAJES BACK-END

* Python.

* PHP.

* Node.js

BASES DE DATOS BACK-END

* MySQL

* MangoDB
